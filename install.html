@gwyzzenn/survey-kit

NPS · CES · CSAT 三合一調查 Popover 套件，供各內部系統引用。一行程式碼觸發問卷，回饋資料自動寫入 Google Sheets。

NPS 0–10 CES 1–7 CSAT 1–5 ★ 零外部依賴 ESM + UMD Cooldown 控制
安裝
三個步驟，工程師約 30 分鐘完成接入
1
設定 .npmrc
在專案根目錄建立，指向 GitHub Packages registry
.npmrc
# 加到專案根目錄的 .npmrc
@gwyzzenn:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_TOKEN

Token 需有 read:packages 權限 → GitHub → Settings → Developer settings → Personal access tokens

2
安裝套件
透過 npm 安裝
bash
npm install @gwyzzenn/survey-kit
3
引入並呼叫
在適當時機觸發問卷，並傳入 onSubmit 送到後端
javascript
import { SurveyKit } from '@gwyzzenn/survey-kit'
import '@gwyzzenn/survey-kit/dist/survey-kit.css'

// 基本用法
SurveyKit.open('NPS')
SurveyKit.open('CES')
SurveyKit.open('CSAT')

// 完整用法
SurveyKit.open('CES', {
  cooldown: 90,
  onSubmit: (data) => {
    fetch('/api/survey', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ ...data, system_id: 'erp' }),
    })
  },
})
API
三個方法，涵蓋所有使用情境
方法參數說明
SurveyKit.open(type, options?)type: 'NPS' | 'CES' | 'CSAT'
options.cooldown?: number（天）
options.onSubmit?: (data) => void		觸發問卷，cooldown 控制同一用戶多少天內不重複顯示
SurveyKit.close(type?)type?: 'NPS' | 'CES' | 'CSAT'手動關閉問卷，不傳 type 則關閉當前顯示中的
SurveyKit.clearCooldown(type?)type?: 'NPS' | 'CES' | 'CSAT'清除冷卻記錄，主要用於開發測試
onSubmit Payload
用戶送出後 onSubmit 收到的資料格式
NPS 範例
{
  type:    "NPS",
  score:   9,
  tags:    ["功能強大", "流程順暢"],
  comment: "整體很好用",
  page:    "/dashboard",
  ts:      "2025-06-10T09:00:00Z"
}
CES / CSAT 範例
{
  type:    "CES",
  score:   3,
  tags:    [],
  comment: "步驟有點多",
  page:    "/checkout",
  ts:      "2025-06-10T09:00:00Z"
}

建議在 onSubmit 時加上 system_id（例如 'erp' / 'crm'）以便後台區分資料來源。

各系統觸發情境
常見的接入方式
javascript
// ERP — 採購流程送出後觸發 CES
async function onOrderSubmit() {
  await submitOrder()
  setTimeout(() => SurveyKit.open('CES', { cooldown: 90, onSubmit }), 1500)
}

// CRM — 客服對話關閉後觸發 CSAT
function onChatClose() {
  SurveyKit.open('CSAT', { cooldown: 30, onSubmit })
}

// 季度 NPS — 登入後自動判斷（cooldown 控制頻率）
window.addEventListener('DOMContentLoaded', () => {
  SurveyKit.open('NPS', { cooldown: 90, onSubmit })
})
注意事項
CSS 必須獨立引入 — import '@gwyzzenn/survey-kit/dist/survey-kit.css' 不可省略，否則樣式不會套用。
.npmrc 不要 commit — 含有 token 的 .npmrc 不應進版控，建議加入 .gitignore，CI 環境改用環境變數注入。
Cooldown 用 localStorage — 同一台機器同一瀏覽器才有效，跨裝置不同步。
